# Mithril.js Basic Template

This is a production-grade starter template for Mithril.js on Urja.

## Adapter
Uses the `mithril-adapter`.

## Development
To start the development server:
```bash
urja dev
```

## Production
To build for production:
```bash
urja build
```

## Features
- VDOM Component in `src/app.js`
- Global styles in `src/styles.css`
- Static assets from `public/`
- Clean uncoupled Mithril mounting logic
