# Alpine.js Basic Template

This is a production-grade starter template for Alpine.js on Urja.

## Adapter
Uses the `alpine-adapter`.

## Development
To start the development server:
```bash
urja dev
```

## Production
To build for production:
```bash
urja build
```

## Features
- Interactive component in `src/app.js`
- Global styles in `src/styles.css`
- Static assets from `public/`
- Zero-core-coupling adapter integration
